# Number Guessing Game 
A simple and fully Python number guessing game where the user has 10 attempts to guess
a randomly generated number between 1 and 1000.It is built using a built-in function named random

## Rules
- The program generates a hidden number
- User has 10 guesses
- Hints are provided after each guess
- Game ends when the number is guessed or attempts are over

## How to Run
```bash
python number_guessing_game.py
